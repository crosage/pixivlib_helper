# Pixiv Helper

Pixiv Helper 

> 目前应用需要 Pixiv session 才能进入主界面。未登录时会停留在登录页。


## 主要功能

- 图库检索：按 tag、作者、PID、收藏数、发布日期等条件筛选作品。
- 网格浏览：图库页和关注页默认使用瀑布流网格，适合快速扫图。
- 关注作品：查看关注作者的新作品，也可以切换到收藏来源。
- 日榜浏览：按 Pixiv 排行榜模式和日期浏览每日榜单。
- 作品详情：查看大图、多 P 切换、作者信息、相关推荐和收藏状态。
- 收藏操作：在卡片、列表、详情页直接收藏或取消收藏。
- 推荐浮层：收藏后展示一组相关作品，方便继续浏览。
- 用户页：管理 Pixiv session、切换用户、查看本地统计摘要。
- 下载管理：保存原图并在下载进度面板里查看任务状态。

## 使用方法

1. 启动应用后会先连接后端服务。
2. 如果没有登录，会进入 Session 登录页。
3. 输入 Pixiv session 或 PHPSESSID，可选填写显示名称。
4. 登录成功后进入主界面。
5. 底部或侧边导航可以切换“图库 / 关注 / 日榜 / 用户”。

获取 Pixiv session 的方式取决于你的浏览器环境。通常可以在已登录 Pixiv 的浏览器开发者工具中查看 Cookie，复制 `PHPSESSID` 或后端要求的完整 session 内容。

## 页面说明

### 图库

图库页用于检索和发现作品。默认是网格模式，可以切换列表模式。筛选条件包括 tag、作者、PID、收藏数区间、发布日期和收藏状态。发现模式会持续加载推荐作品，并自动过滤已经看过的 PID。

### 关注

关注页用于查看关注作者的新作品，也可以切换收藏来源。页面支持 tag 和作者过滤，默认使用网格模式。

### 日榜

日榜页用于查看 Pixiv 排行榜。可以切换榜单模式、翻页和选择日期。点击作品会进入详情页。

### 用户

用户页用于管理当前 session、切换已登录用户、查看系统统计和常用 tag。这个页面会加载用户 session、系统摘要、tag 统计和当前 Pixiv 用户信息，因此首次进入可能比其他页面更慢。

## 运行

安装 Flutter 依赖后运行：

```bash
flutter pub get
flutter run
```

默认后端地址在代码中配置为 `https://api.zundamon.bond`。如果要连接自己的后端，可以通过 dart define 覆盖：

```bash
flutter run --dart-define=PIXIV_HELPER_API_BASE=http://127.0.0.1:8000
```

## 缓存

图片使用本地缓存。Windows 下默认缓存目录为：

```text
D:\PixivHelperCache\image_cache
```

缓存可以提升二次浏览速度，但首次打开 Pixiv 图片仍然取决于后端代理和 Pixiv 图片源的网络状况。

## 注意事项

- 应用依赖后端接口，单独运行前端无法获取 Pixiv 数据。
- Pixiv 图片会经过后端代理，以便携带必要请求头和处理跨域/防盗链问题。
- 如果图片加载慢，优先检查后端代理到 Pixiv 的网络、图片缓存命中率，以及当前页面是否正在预取大量图片。
